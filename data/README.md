# Data folder

Place your downloaded datasets here. **Do not commit large CSV files to Git.**

| File | Task | Download from |
|------|------|----------------|
| `fraudTrain.csv` | Task 1 – Fraud | [Kaggle – Fraud Detection](https://www.kaggle.com/datasets/kartik2112/fraud-detection) |
| `fraudTest.csv` | (optional) | Same dataset |
| `Churn_Modelling.csv` | Task 2 – Churn | [Kaggle – Bank Churn](https://www.kaggle.com/datasets/shantanudhakadd/bank-customer-churn-prediction) |
| `spam.csv` | Task 3 – SMS Spam | [Kaggle – SMS Spam](https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset) |

Then run from the project root: `python fraud_detection.py`, `python churn_prediction.py`, `python sms_spam_detection.py`.
